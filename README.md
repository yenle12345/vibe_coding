Mô tả bài toán:
Tạo một web app quản lý sinh viên đơn giản.
Người dùng có thể:
1.	Thêm sinh viên
2.	Xem danh sách sinh viên
3.	Sửa thông tin sinh viên
4.	Xóa sinh viên
Thay đổi nghiệp vụ & mở rộng chức năng
Sau khi MVP chạy được, yêu cầu chỉnh sửa hệ thống: thêm lớp học, thay đổi nghiệp vụ(1 sinh viên có 1 lớp học, một lớp học có nhiều sinh viên), thống kê bảng dữ liệu sinh viên, cho phép xuất file csv.
